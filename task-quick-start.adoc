:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Per iniziare a utilizzare Azure NetApp Files, configura un'applicazione Microsoft Entra e crea un sistema.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Uno"] link:task-set-up-azure-ad.html["Configurare un'applicazione Microsoft Entra"]
[role="quick-margin-para"]
Da Azure, concedi le autorizzazioni a un'applicazione Microsoft Entra e copia l'ID dell'applicazione (client), l'ID della directory (tenant) e il valore di un segreto client.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Due"] link:task-create-system.html["Creare un sistema Azure NetApp Files"]
[role="quick-margin-para"]
Dalla pagina Sistemi nella NetApp Console, seleziona *Aggiungi sistema* > *Microsoft Azure* > * Azure NetApp Files*, quindi fornisci i dettagli sull'applicazione Active Directory.
